## Overview

**Parallel and Distributed Computing** is a course offered in the second semester of the third year of the LEIC Bachelor's program at the Faculty of Engineering, University of Porto.

This repository contains **two independent projects**:

1. **Multiplication Analysis**  
   Analysis of different multiplication algorithms, including:  
   - Normal multiplication  
   - Linear multiplication  
   - Block multiplication  
   - Parallel multiplication  
   Focus on performance evaluation and algorithmic efficiency. Implemented in **C++** and **Java**.

2. **Hangman Game with Concurrency**  
   A simplified Hangman game where players guess the word by letters or the full word.  
   Features include:  
   - Fault tolerance in concurrent environments  
   - User authentication and ranking system  
   - Points system: +30 for a win, -10 for a loss  
   - Handling multiple players and concurrent access  
   Implemented in **Java**.
