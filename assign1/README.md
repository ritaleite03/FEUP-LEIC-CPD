# Multiplication Analysis – Project 1

**Multiplication Analysis** evaluates and compares different multiplication algorithms in terms of **performance and efficiency**, implemented in **C++** and **Java**.

> Focus on algorithmic efficiency and performance benchmarking with various input sizes.

## 📌 Algorithms Implemented

| Algorithm                   | Description                                                   |
| --------------------------- | ------------------------------------------------------------- |
| **Normal Multiplication**   | Standard grade-school method                                  |
| **Linear Multiplication**   | Sequential, optimized for smaller inputs                      |
| **Block Multiplication**    | Divides matrices into sub-blocks to improve cache performance |
| **Parallel Multiplication** | Multi-threaded computation for faster execution               |

## ⚡ Features

- Compare **execution time** across input sizes
- Analyze **parallel speedup** using multi-threading
- Supports **large matrices** efficiently
